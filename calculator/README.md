# Calculadora Rosa

Calculadora visual estilizada com tema rosa pastel, construída com Java Swing.

---

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Funções auxiliares: porcentagem (`%`), troca de sinal (`±`), backspace (`←`) e clear (`C`)
- Display customizado com fonte grande e alinhamento à direita
- Design responsivo com cores diferenciadas por tipo de botão

---

## Design

- **Tema rosa pastel** com fundo suave (`#FAEBF3`)
- **Botão `=`**: destaque em rosa forte (`#FF69B4`) com texto branco
- **Operadores (`+`, `-`, `*`, `/`)**: rosa médio (`#FF8CB4`)
- **Funções (`C`, `←`, `%`, `±`)**: rosa claro (`#FFDCE6`)
- **Números**: rosa padrão (`#FFBED2`)

---

## Como Executar

```bash
cd calculator
javac Main.java
java Main
```

---

## Requisitos

- Java JDK 8 ou superior
